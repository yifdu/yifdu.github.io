数字图像处理笔记（三）

发表于 2018-12-21 | 分类于 Computer Vision | 阅读数 次

彩色基础知识

  • 彩色图像处理可分为：
  1. 全彩色处理：数码相机、数码摄像机、彩色扫码仪
  2. 伪彩色处理： 对不同的灰度或灰度范围赋予不同的颜色
  • 光是颜色科学的核心
  • 描述彩色光的3个基本量：
  1. 辐射率：从光源流出能量的总量，用瓦特(W)度量
  2. 光强:观察者从光源接收的能量总和
  3. 亮度:主观描绘子

彩色空间

  • RGB
  1. CCD技术直接感知RGB三个分量
  2. 是图像成像、显示、打印等设备的基础
  • CMY 和CMYK
    CMY(青、深红、黄)、CMYK(青、深红、黄、黑)
  • HSI(色调、饱和度、亮度)
  1. 两个特点：I分量与图像的彩色信息无关；H和S分量分量与人感受颜色的方式是紧密相连的
  2. 将亮度(I)和色调(H)和饱和度(S)分开
  3. 避免颜色受到光照明暗(I)等条件的干扰
  4. 仅仅分析反映色彩本质的色调和饱和度
  5. 广泛用于计算机视觉、图像检索和视频检索
  • YIQ
  1. Y指亮度(Brightness),即灰度值
  2. I和Q指色调，描述色彩及饱和度
  3. 用于彩色电视广播，被北美的电视系统所采用(属于NTSC系统)
  4. Y分量可提供黑白电视机的所有影像信息
  • YUV
  1. Y指亮度，与YIQ的Y相同
  2. U和V也指色调，不同于YIQ的I和Q
  3. 用于彩色电视直播
  4. Y分量也可提供黑白电视机的所有影响信息
  • YCbCr
  1. Y指亮度，与YIQ和YUV的Y相同
  2. Cb和Cr由U和V调整
  3. JPEG采用的彩色空间

伪彩色处理

什么叫伪彩色图像处理？
根据一定的准则对灰度值赋以彩色处理
区分：伪彩色图像、真彩色图像、单彩色图像
为什么需要伪彩色处理?
人类可以辨别上千种颜色和强度，只能辨别20多种灰度
应用：为人们观察和解释图像种的灰度目标
怎样进行伪彩色图像处理:
1.强度分层技术
2.灰度到彩色转换技术

强度分层技术

  • 把一幅图像描述为三维函数(x,y,f(x,y))
  • 分层技术:放置平行于(x,y)坐标面的平面
  • 每一个平面在相交区域切割图像函数

    定义：令[0,L-1]表示灰度级，使$l_0$表示黑色(f(x,y)=0),$l_{L+1}$表示白色(f(x,y)=L-1).假设垂直于强度轴的P个平面定义为量级$l_1,l_2,…,l_p$.0＜P＜L-1,P个平面将灰度级分为P+1个间隔，$V_1,V_2,…,V_{P+1}$，则灰度级到彩色的赋值关系:$c_k$是与强度间隔$V_k$第K级强度有关的颜色
    $V_k$是由在l=k-1和l=k分割平面定义的

例如甲状腺模型

作图的恒定强度难以区分病变，右图强度分层结果，清楚的显示恒定强度的不同区域。
例如用颜色突出降雨水平

灰度到彩色的转换

  • 对任何输入像素的灰度级执行3个独立变换
  • 3个变换结果分别送入彩色监视器的红、绿、蓝三个通道
  • 产生一幅合成图像

全彩色图像处理

全彩色图像处理研究分为两大类:

  • 分别处理每一分量图像，然后合成彩色图像
  • 直接对彩色像素处理:3个颜色分量表示像素向量。令c代表RGB彩色空间中的任意向量

彩色变换

彩色变换函数
g(x,y)=T[f(x,y)]
其中f(x,y)是彩色输入图像
g(x,y)是变换或处理过的彩色输出图像
T是在空间邻域(x,y)上对f的操作
彩色变换的简单形式

$r_i$和$s_i$是f(x,y)和g(x,y)在任何点处彩色分量的变量
${T_1,T_2,…,T_n}$是一个对$r_i$操作产生$s_i$的变换或彩色映射函数集
选择的彩色空间决定n的值，如RGB彩色空间，n=3,
$r_1,r_2,r_3$表示红、绿、蓝分量；CMYK,则n=4

补色:在如图所示的彩色环上，与一种色调直接相对立的另一种色调称为补色

作用:增强嵌在彩色图像暗区的细节
